[Техническая документация](https://dmta-mlita-project-version-nativejs.readthedocs.io/en/latest/)

## Установка и запуск

Требования:
> Установленный Node.js, рекомендуемая версия: v16.15.0

Запустить DMTA-Solver можно командами:

```shell
npm install
```

```shell
npx serve
```
